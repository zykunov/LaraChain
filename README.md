# Блокчейн‑приложение на Laravel

**Блокчейн‑платформа для безопасных транзакций** на базе фреймворка Laravel. Проект реализует упрощённую версию блокчейна с возможностью создания блоков, добавления транзакций и проверки целостности цепочки.

## Функциональность

* Создание новых блоков с транзакциями.
* Добавление транзакций в ожидающую очередь.
* Расчёт и проверка хешей блоков (SHA‑256).
* Проверка целостности всей цепочки блоков.
* API для взаимодействия с блокчейном.
* Веб‑интерфейс для просмотра блоков и транзакций.
* Логирование операций с блокчейном.

## Технологии

* **Backend:** Laravel 12
* **Front-end:** Vue.Js
* **База данных:** MySQL/PostgreSQL
